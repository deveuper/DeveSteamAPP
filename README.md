## 📥 下载安装

### 方式一：从 Release 下载（推荐）

👉 **[前往 Releases 页面](https://github.com/deveuper/DeveSteamAPP/releases)** 👈

1. 下载最新版本的 `deve-steamdeck-toolbox.zip`
2. 将压缩包解压到 `~/homebrew/plugins/` 目录
3. 重启 Steam Deck

### 方式二：一键安装脚本

在桌面模式打开终端，运行：

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/deveuper/DeveSteamAPP/main/install-guide/install.sh)"
```

### 方式三：安装引导页面

访问 **[安装引导页](https://deveuper.github.io/DeveSteamAPP)** 查看图文教程

## 📋 安装要求

- **硬件**：Steam Deck（SteamOS）
- **前置**：已安装 [Decky Loader](https://github.com/SteamDeckHomebrew/decky-loader)
- 如果未安装 Decky，工具箱内提供**一键安装**功能
