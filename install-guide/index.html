📦 3 步安装

1

切换到桌面模式

按住电源键 → 选择「切换到桌面」

2

下载安装包

点击下方按钮下载最新版本安装包

⬇️ 下载最新版本
3

安装并使用

解压安装包到 Decky 插件目录，或使用一键安装脚本：

sh install.sh

安装完成后切回游戏模式，在 Decky 中即可看到工具箱

⚡ 一键安装（进阶）

在桌面模式终端中运行以下命令，自动完成安装：

sh -c "$(curl -fsSL https://raw.githubusercontent.com/your-repo/SteamdeckAPP/main/install-guide/install.sh)"

📱 手机扫码安装

用手机扫描下方二维码，在手机上查看安装教程

❓ 常见问题

需要先安装 Decky Loader 吗？

是的，本工具箱是 Decky Loader 插件。如果还没安装 Decky，可以在工具箱内一键安装，或访问 Decky Loader 官方页面。

安装后在哪里找到工具箱？

安装完成后，切回游戏模式，按快捷键打开 Decky 菜单（通常为 ≡ 按键），在插件列表中即可看到「Steam Deck 工具箱」。

修改密码安全吗？

安全。密码通过系统命令 chpasswd 修改，不会记录在日志中，传输过程也不经过网络。
